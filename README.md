# 🧠 ML Atlas — Guía Interactiva de Machine Learning

Guía interactiva y visual para aprender, entender y recordar cuándo usar cada modelo y herramienta de Machine Learning.

## 🔗 [Abrir la guía →](https://sebachamorrov.github.io/ml-atlas/)

---

## ¿Qué incluye?

**33 modelos, algoritmos y herramientas** organizados en 5 categorías:

| Categoría | Cantidad | Ejemplos |
|-----------|----------|----------|
| Supervisado | 11 | Linear Reg, Logistic Reg, Decision Tree, Random Forest, XGBoost, LightGBM, CatBoost, SVM, KNN, Naive Bayes |
| No Supervisado | 6 | K-Means, DBSCAN, HDBSCAN, PCA, Apriori, FP-Growth |
| Series de Tiempo | 3 | ARIMA, SARIMA, Prophet |
| Deep Learning | 5 | MLP, CNN, RNN, LSTM/GRU, Transformer |
| Herramientas DS | 8 | GridSearchCV, RandomizedSearchCV, Optuna, LazyPredict, Cross-Validation, SHAP, SMOTE, Sklearn Pipeline |

## Funcionalidades

- **Flowchart de decisión paso a paso** — respondé preguntas y llegá al modelo correcto
- **Simuladores visuales interactivos** — mové parámetros y observá cómo cambia el modelo en tiempo real (Linear Reg, Logistic Reg, Decision Tree, KNN, K-Means, DBSCAN, ARIMA, MLP, Transformer)
- **Buscador inteligente de pipeline** — 4 preguntas → pipeline completo con modelo, tuning y preprocesamiento adaptado a tu caso
- **Explicación completa por modelo** — qué es, cómo funciona, ventajas, limitaciones y cuándo usarlo

## Cómo usar

Es un archivo HTML standalone. Dos opciones:

1. **Online:** abrí el [link de GitHub Pages](https://sebachamorrov.github.io/ml-atlas/)
2. **Local:** descargá `index.html` y abrilo en cualquier navegador

No necesita servidor, base de datos, ni instalación.

## Tecnologías

HTML + CSS + JavaScript vanilla. Sin dependencias externas (excepto Google Fonts). Los simuladores usan Canvas API.
